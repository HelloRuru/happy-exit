# 🎉 離職全能導航幫手 (Happy Exit)

> 新的開始，從溫暖的告別開始 ✨

一個幫助台灣勞工順利離職的互動式工具，涵蓋法律權益計算、離職信生成、交接清單管理等功能。

## 📋 功能總覽

### 📌 第一部分：法律權益與行政防線

| 功能 | 說明 |
|------|------|
| **預告期計算** | 依《勞基法》自動計算需要的預告天數 |
| **謀職假精算器** | 自動計算預告期間每週 2 天的有薪謀職假，並產生申請單範本 |
| **特休結算表** | 曆年制/週年制換算，精確計算應折算為薪資的剩餘天數 |
| **非自願離職代碼檢查** | 對照勞基法第 11 條，確保離職證明書代碼符合失業補助資格 |
| **資遣費試算** | 串接勞動部公式，計算年資折算金額 |
| **服務證明書提醒** | 依勞基法第 19 條，確認證明書內容無不利記載 |
| **勞健保轉出單確認** | 提醒確保下一間公司或公所投保銜接 |
| **勞退 6% 提繳檢查** | 引導至勞保局系統檢查最後提繳紀錄 |
| **競業禁止條款效力自檢** | 檢查合約是否有「月補償金」約定 |
| **數位薪資單備份提醒** | 提醒下載近兩年 PDF 作為報稅憑證 |

### 📌 第二部分：數位安全與體面轉身

| 功能 | 說明 |
|------|------|
| **公司信箱清理指南** | 關鍵字搜尋刪除（面試、薪資、體檢、私人）+ Hard Delete |
| **外部服務帳號換綁** | 將 Notion、Figma、Canva 等工具改綁個人信箱 |
| **作品集去識別化導航** | 數據轉百分比，確保不違反保密協議 |
| **方法論 SOP 備份** | 保留「技能」而非「公司資料」 |
| **瀏覽器清理** | 登出帳號、清除密碼、Cookies |
| **交接檔案命名指南** | `YYYYMMDD_專案名_交接版` 結構化命名 |

### 📌 第三部分：文件與範本生成

| 功能 | 說明 |
|------|------|
| **離職信生成器** | 正式/親切/簡潔三種風格 |
| **Email 範本** | 主管面談、人資通知、同事道別、廠商告知 |
| **Farewell Mail 多版本** | 依對象分類的離職信範本 |
| **推薦信請求話術** | 向主管請求 LinkedIn 推薦的優雅說法 |
| **LinkedIn 動態範本** | 體面的中英文離職宣告範本 |

### 📌 第四部分：交接清單

- 30+ 項完整檢查清單
- 分類：法律權益、工作交接、數位安全、物品歸還、人際關係
- 優先度標記：必做 / 建議 / 可選
- **可輸出格式**：
  - Excel (CSV)
  - Word (HTML)
  - Google Docs（開啟新建頁面）
  - Google Sheets（開啟新建頁面）

## 🚀 快速開始

```bash
# 安裝依賴
npm install

# 開發模式
npm run dev

# 建置生產版本
npm run build

# 預覽生產版本
npm run preview
```
